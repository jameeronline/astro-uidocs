## Quick Start

Get up and running with Park UI in minutes:

<CardGrid>
  <Card title="1. Install" icon="add-document">
    ```bash
    npm install park-ui
    ```
  </Card>
  
  <Card title="2. Import Styles" icon="setting">
    ```css
    @import 'park-ui/styles';
    ```
  </Card>
  
  <Card title="3. Use Components" icon="rocket">
    ```jsx
    import { Button } from 'park-ui';
    
    <Button variant="primary">
      Get Started
    </Button>
    ```
  </Card>
</CardGrid>
